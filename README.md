# 🌍 AI Language Translation Tool

## 📌 Overview
This is an AI-powered Language Translation Tool built using Python, Streamlit, and Deep Translator. It translates text into multiple languages with a simple and user-friendly interface.

## 🚀 Features
- 🌐 Translate into 20+ languages
- 🔍 Auto language detection
- 🔄 Select source and target languages
- 🎨 Clean Streamlit interface
- ⚡ Fast and easy to use

## 🛠️ Technologies Used
- Python
- Streamlit
- Deep Translator

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/tsachitpatra-creator/CodeAlpha-Language-Translator.git
```

Install the required libraries:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
